# 🚀 Pixora - MERN Social Media App

> A modern full-stack social media app where users can upload images, add captions, and explore a dynamic feed.

---

## 🌟 Features

✨ Upload images with captions  
📸 Beautiful responsive feed UI  
⚡ Fast API with Express  
🗂️ Organized full-stack structure  
🎨 Modern glassmorphism design  

---

## 🛠️ Tech Stack

### 🔹 Frontend
- React (Vite)
- Axios
- Custom CSS (Glass UI)

### 🔹 Backend
- Node.js
- Express.js
- MongoDB
- Multer (Image Upload)
- ImageKit / Cloud Storage (if used)
